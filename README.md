Ця програма розроблена для ведення невеликого цифрового каталогу літератури з використанням масиву спеціальних структур даних. За допомогою графічного інтерфейсу користувач вводить інформацію про автора, назву та рік публікації книги, яку програма перевіряє та безпечно зберігає в оперативній пам'яті. Щоб забезпечити наочність процесу, кожен успішно збережений запис одразу відображається у загальному текстовому списку на головному екрані. Ключовою функцією програми є фільтрація накопичених даних: спеціальний алгоритм перевіряє всі книги та відбирає виключно ті, що були надруковані раніше заданого контрольного року. Знайдені результати пошуку, як і довідкові відомості про розробника та лабораторну роботу, демонструються у зручних спливаючих вікнах, які легко викликати через кнопки або навігаційне меню форми.
